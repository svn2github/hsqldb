It is recommended to run the shell scripts(from /bin and /demo) from the console,
as you will not to be able see any messages from the desktop.
you should set the $JAVA_HOME before running any of the sripts
or you can use the -jdkhome switch instead to specify the JDK.

startup scripts can be found in /bin,
You will need to modify hsqldbserver and hsqldbwebserver before copying
the files into /etc/init.d/
