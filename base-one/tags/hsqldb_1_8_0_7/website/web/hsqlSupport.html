If you have a question ...

As a first step, please read the How To / FAQ (Frequently Asked Questions) and the documentation.

If you still need support, you have two options:

Alternative A

The Help Forum is a web-based discussion board for support questions.

Read and search the archives of the Help Forum

And if you cannot find the answer, try Alternative B below before posting a new message either in an existing, related, thread, or in a new thread. You should get a SourceForge user-id and logon before you submit items. You can then put bookmarks on threads that interest you or choose to get the new posts by email.

DO NOT post in the Open Discussion Forum unless you want to discuss future developments of HSQLDB.

Alternative B

Alternatively, you can use our user mailing list. You can read and search the archives before posting a question. You must subscribe to the mailing list before you can post a message.

Read and search the archives of the Mailing List.
Read and search the archives of the Mailing List at gmane.org (better search facitily).
Subscribe to this mailing list

Register a Bugs / Feature Requests

If you are new to Java or databases, please discuss the issue in one of the forums first. If you have to report a bug in HSQLDB please use the Bug Tracker system at SourceForge. If possible, include a test case or the code that caused the error. The error can then be fixed much easier and faster.

Use the Feature Request Tracker to post a new feature request.

It is a good idea to get a SourceForge user-id and logon before you submit items. This will ensure that you get an email if there is a follow-up or a fix.
