HSQLDB 1.7.2 ALPHA_E

Strictly for developers. This is alpha software featuring developments
leading to version 1.7.2

Use of Vector, Hashtable and StringBuffer has been replaced by
new HSQLDB collection classes (in the org.hsqldb.lib package).
