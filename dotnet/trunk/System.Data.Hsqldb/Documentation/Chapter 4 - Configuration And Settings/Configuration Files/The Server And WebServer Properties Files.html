Chaper Overview > Configuration Files > The Server And WebServer Properties Files

Common Properties

Value Default Description
server.database.0 test The actual path and file name of the default database
server.dbname.0 "" URL path alias of the default database
server.urlid.0 NONE

SqlTool urlid used by UNIX init script.

This property is not used if your are running Server/Webserver on a platform other than UNIX, or of you are not using the HSQLDB UNIX init script
server.silent true Inhibits writing verbose message generation to the standard output
server.trace false Enables writing trace level messages to the standard output

Notes

Presently, each Server or WebServer can host up to a maximum of 10 database instances simultaneously.

The server.database.0 property is special in that it denotes the default database, i.e. the one corresponding to the empty path: "Path=/;" or "Path=;" or simply path omitted.

Values for server.database.{0-9} properties accept the mem:, file: or res: URI scheme prefixes coresponding to the three supported In-Proccess database protocols and may include trailing database-specific properties.

For example:

    database.0=mem:temp;sql.enforce_strict_size=true;sql.tx_no_multi_rewrite=true;

Server Properties

Value Default Description
server.port 9001 TCP/IP port used for talking to clients. All databases are served on the same port.
server.no_system_exit true false enables the System.exit() call when the server is stopped.

WebServer Properties

Value Default Description
server.port 80 The TCP/IP port used for talking to clients
server.default_page index.html The default web page for server
server.root ./ The location of served pages
.<extension> N/A The mime type associated with the file extension of static files served by the web server.

Notes

All the properties above can be specified on the server command line by replacing the server. prefix with a dash and the equals symbol by a space:

    >System.Data.Hsqldb.Server.exe -database.0 file:c:\test -dbname.0 myapplication -port 12345 -silent false -trace true
