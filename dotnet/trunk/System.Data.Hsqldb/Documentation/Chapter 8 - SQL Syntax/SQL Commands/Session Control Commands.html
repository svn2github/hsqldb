Chapter 8 - SQL Syntax
Chapter Overview > SQL Commands > Session Control Commands

CONNECT

CONNECT USER <username> PASSWORD <password>;

Password should be delimited using leading and trailing double quote characters.

Use "" to denote the empty password.

DISCONNECT

DISCONNECT;

Closes the current session.

After disconnecting, it is not possible to execute other queries (including CONNECT).

This command should be reserved for SQL scripts only.

From ADO.NET, the System.Data.IDbConnection.Close() method should be used instead.

SET PASSWORD

SET PASSWORD <password>;

Changes the password of the currently connected user.

Password should be delimited using leading and trailing double quote characters.

Use "" to denote the empty password.

SET SCHEMA

SET SCHEMA <schemaname>;

Sets the default schema qualifier for the current session.

This is the schema used to qualify database object identifiers when the schema name is not provided explicitly.

The initial value is PUBLIC.
