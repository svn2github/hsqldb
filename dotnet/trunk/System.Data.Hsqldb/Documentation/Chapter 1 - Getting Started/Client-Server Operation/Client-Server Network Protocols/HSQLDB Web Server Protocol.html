Chapter 1 - Getting Started
Chapter Overview > Client-Server Operation > Client-Server Network Protocols > HSQLDB Web Server Protocol 

Use when the computer hosting the database server is restricted to the HTTP protocol.

The only reason for using the HSQLDB Web Server is to overcome firewall restrictions imposed on client or server machines.

It should not be used when there are no such restrictions.

An HSQLDB WebServer process may be started by running the provided System.Data.Hsqldb.WebServer.exe console application, for example:

    >System.Data.Hsqldb.WebServer.exe -database.0 file:c:\dbfiles\app1 -dbname.0 appdata

Note

The command line argument -? can be used to get a list of available arguments.
