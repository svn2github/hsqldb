Chapter Overview > Client-Server Operation > Client-Server Network Protocols

The HSQLDB ADO.NET Data Provider supports two main client-server network protocols:

  • HSQL native protocol over TCP/IP.
  • HSQL native protocol tunneled over HTTP.
