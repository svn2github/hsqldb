Chapter 1 - Getting Started
Chapter Overview > Client-Server Operation > Client-Server Network Protocols > HSQLDB Server Protocol

This is the preferred (and most performant) way of providing client-server access to a database instance.

The HSQLDB Server uses a proprietary communications protocol (HSQL).

An HSQLDB Server process may be started by running the provided System.Data.Hsqldb.Server.exe console application, for example:

    >System.Data.Hsqldb.Server.exe -database.0 file:c:\dbfiles\db1 -dbname.0 xdb

Note

The command line argument -? can be used to get a list of available arguments.
