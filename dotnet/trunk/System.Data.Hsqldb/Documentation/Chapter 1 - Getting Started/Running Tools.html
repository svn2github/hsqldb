Chapter 1 - Getting Started
Chapter Overview > Running Tools

All tools can be run in the standard way, either from the console, a shortcut, or by double clicking directly on the executable.

In the following example, the Database Manager executable is located in the current directory:

    >System.Data.Hsqldb.DatabaseManager.exe

Main Hsqldb tools

  • System.Data.Hsqldb.DatabaseManager.exe
  • System.Data.Hsqldb.SqlTool.exe
  • System.Data.Hsqldb.Server.exe
  • System.Data.Hsqldb.WebServer.exe

Note

All tools can use command line arguments, and SqlTool absolutely relies upon them.

You can add the command line argument -? to get a list of available arguments for each tool.

The Database Manager features a graphical user interface and can be used to explore a database interactively.
