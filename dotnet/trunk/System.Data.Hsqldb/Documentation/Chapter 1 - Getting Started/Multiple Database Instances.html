Chapter 1 - Getting Started
Chapter Overview > Multiple Database Instances

In the examples outlined in the preceding sections, each server hosts only one database instance and only one database instance is used per application.

However, each HSQLDB Server instance can actually host several database instances (currently limited to 10) and there is no programmatically enforced limit on the number of In-Process database instances that an application may open.

Similarly, there is no programmatic limit on the number of connections that may be made to each database instance.

Note

These capabilities are covered to greater depth in Chapter 4 - Configuration And Settings.
