Chapter 2 - SQL Issues
Chapter Overview > Types And Arithmetic Operations > Bit And Boolean Types

As per the SQL:2003 standard, the HSQLDB SQL BIT type is deprecated as of release 1.8 and is now simply an alias for SQL BOOLEAN.

The primary representation of SQL BOOLEAN column is TRUE or FALSE, either as the boolean type or as an SQL string literal.

The SQL BOOLEAN type can be initialized using values of any numeric type, where 0 is translated to FALSE and any other value is translated to TRUE.

The HSQLDB SQL BOOLEAN type conforms to the latest SQL standard (SQL:2003) and supports the UNDEFINED state in addition to TRUE or FALSE.

NULL values are treated as UNDEFINED. This improvement affects queries that contain NOT IN.
